## 1. ISO/IEC 27001:2022 Պաշտոնական սերտիֆիկացում

Համաձայն պաշտոնական միջազգային համապատասխանության սերտիֆիկատի, որը տրվել է անկախ աուդիտոր *Schellman Compliance, LLC*-ի կողմից **2025 թվականի դեկտեմբերի 12-ին**, Microsoft Azure-ի տեղեկատվական անվտանգության կառավարման համակարգը (ISMS) լիովին համապատասխանում է **ISO/IEC 27001:2022** ստանդարտին:

> **Տվյալներ սերտիֆիկատից՝**
>
> * **Սերտիֆիկատի համարը**՝ 1729711-17
> * **Գործողության ոլորտը (Scope)**՝ Azure Public Cloud-ի մշակում, շահագործում և ենթակառուցվածքներ

Մեր թիմի կողմից նախագծի իրականացման համար ընտրված ծառայությունները պաշտոնապես ներառված են սերտիֆիկացված տիրույթում (In-Scope Services)՝

* 🔹 **Microsoft Foundry: Azure Language** (Պատասխանատու է անձնական տվյալների՝ PII ատրիբուտների արտահանման համար)
* 🔹 **Microsoft Foundry: Azure Speech** (Օգտագործվում է աուդիոյի տրանսկրիպցիայի համար, եթե կիրառելի է)

## 2. Տվյալների տեղակայում (Data Residency)

Ընկերության գաղտնիության պահանջներին և ISO 27001 սկզբունքներին համապատասխանելու համար Azure AI-ի բոլոր ռեսուրսները տեղակայվում են բացառապես **West Europe (Արևմտյան Եվրոպա)** տարածաշրջանում:

Համաձայն սերտիֆիկատի հավելվածի (էջ 18–21), Microsoft-ի եվրոպական տվյալների կենտրոններն անցել են անվտանգության ամբողջական ստուգում՝

* **Մշակման հիմնական հանգույցը**՝ Ամստերդամ, Նիդերլանդներ (*Amsterdam, Netherlands* — In-Scope Datacenter)

## 3. Տվյալների մշակման և անվտանգության կանոններ

Տվյալների մշակման ողջ գործընթացում ապահովվում է բարձր մակարդակի գաղտնիություն և պաշտպանվածություն հետևյալ մեթոդներով՝

* **Տվյալների փոխանցման անվտանգություն** — Օգտագործողի և Azure AI ծառայության միջև ողջ տեղեկատվությունը փոխանցվում է բացառապես շիֆրավորված TLS 1.2 կամ TLS 1.3 արձանագրությունների միջոցով։ Սա բացառում է տրաֆիկի գաղտնալսումը երրորդ անձանց կողմից։
* **Տվյալների ժամանակավոր պահպանում** — Եթե համակարգում առաջանում են ժամանակավոր քեշավորված տվյալներ, դրանք ավտոմատ կերպով շիֆրավորվում են սերվերային մակարդակում՝ AES-256 ստանդարտով։
* **Stateless PII մշակում** — Մեր ընտրած Azure AI Language API-ն աշխատում է առանց տվյալների վիճակի պահպանման (Stateless) ռեժիմով։ Տեքստից անունները, հասցեները և այլ անձնական տվյալները (PII) արտահանելուց և պատասխան JSON-ն ուղարկելուց հետո, բոլոր լոգերն ու տվյալները վայրկյանական ջնջվում են օպերատիվ հիշողությունից։ Տվյալները չեն օգտագործվում Microsoft-ի կողմից արհեստական բանականության մոդելների ուսուցման համար։
